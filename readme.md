# wp4-cli

A cli-tool based on webpack4.

## Installation
Using npm:
```
npm install -g wp4-cli
```
## Usage
Run command:
```
wp4-cli init
```
Then the cli will generate files in the current directory,including **webpack.config.js**„ÄÅ**index.js** etc.

## Develop
Run command:
```
npm run start
```
Open your broswer and visit `http://localhost:8080/`

## Build
Run command:
```
npm run build
```
